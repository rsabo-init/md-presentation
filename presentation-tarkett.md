## Project 1: Tarkett PVC

**SCADA for Advanced Production Supervision**

- **Started:** 2016
- **Approach:** Iterative development with continuous improvements
- **Scope:** Full production line monitoring and control

**Key Deliverables:**
- Real-time monitoring of all key production parameters
- Command system for fast adjustment of production settings
- Full ERP integration with automated data exchange
- Process digitalization for increased transparency

# Image Placeholders for SCADA Presentation

This folder contains images used in the presentation. 

## Required Images:

### 1. scada-overview.png
- **Slide:** Title slide
- **Suggested content:** SCADA system overview diagram or water facility photo
- **Recommended size:** 1200x600px

### 2. system-architecture.png
- **Slide:** System Architecture
- **Suggested content:** Technical diagram showing RTUs, communication network, and control room
- **Recommended size:** 1400x800px

### 3. monitoring-dashboard.png
- **Slide:** Monitoring Capabilities
- **Suggested content:** Screenshot of SCADA HMI dashboard with real-time data
- **Recommended size:** 1400x800px

### 4. gis-map.png
- **Slide:** Distributed Location Management
- **Suggested content:** GIS map showing water distribution network with markers
- **Recommended size:** 1400x700px

## How to Add Images:

1. Place your image files in this folder with the names above
2. Or rename the examples in presentation.md to match your file names
3. The presentation will automatically display them

## Image Format Tips:

- Use PNG for diagrams and screenshots (better quality)
- Use JPG for photos (smaller file size)
- Recommended dimensions: 1200-1600px wide for best quality
- Keep file sizes under 1MB for faster loading

## Example Markdown Syntax:

```markdown
![Image Description](images/your-image.png)
```

Replace "your-image.png" with your actual filename.
